# Azure DevOps Sprint Analytics Dashboard

Interactive sprint and backlog analytics dashboards built using Azure DevOps Analytics to monitor team performance, work distribution, and sprint execution metrics.

---

## Overview

This project showcases sprint-focused dashboards created within Azure DevOps to provide real-time insights into agile team performance and operational metrics.

The dashboards were designed to:

- Track sprint progress
- Monitor velocity and burndown
- Analyze backlog distribution
- Monitor bug trends
- Evaluate work item flow
- Provide visibility into team productivity

---

## Dashboard Highlights

### Sprint Monitoring
- Active bugs tracking
- Tasks in progress
- Work items moved out
- Planned vs completed work
- Impediments monitoring
- Current sprint workload

### Agile Performance Metrics
- Sprint burndown charts
- Average velocity tracking
- Work item completion percentage
- Scope increase tracking
- Stories remaining

### Backlog & Work Distribution
- Backlog state distribution
- Work item type breakdown
- Items created and updated
- Mentioned and followed items
- New work (< 7 days)

### Team Analytics
- Work item velocity across iterations
- Backlog growth over time
- Active vs committed work
- Team workload visualization
